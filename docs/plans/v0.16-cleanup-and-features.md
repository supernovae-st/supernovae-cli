### Task 2.2: Storage API Exposure

**Current state:** `storage/local.rs` has 10 reserved API methods never called.

**Methods to integrate:**
- `root()` вҶ’ Storage root path
- `packages_dir()` вҶ’ Packages directory
- `cache_dir()` вҶ’ Cache directory
- `is_installed(name)` вҶ’ Check if package installed
- `get_installed(name)` вҶ’ Get installed package info
- `list_installed()` вҶ’ List all installed packages
- `invalidate_cache()` вҶ’ Clear package cache
- `clear_cache()` вҶ’ Remove cache directory

**Integration points:**
1. `spn list` вҶ’ Use `list_installed()`
2. `spn info <pkg>` вҶ’ Use `get_installed()`
3. `spn cache clear` вҶ’ Use `clear_cache()`
4. `spn doctor` вҶ’ Use `is_installed()` checks

### Task 2.3: Confirmation Prompts Integration

**Current state:** `prompts.rs` has `confirm_delete()` and `confirm_overwrite()` but never used.

**Integration:**
1. `spn remove <pkg>` вҶ’ Call `confirm_delete()` before removal
2. `spn install --force` вҶ’ Call `confirm_overwrite()` before overwriting
3. `spn mcp remove` вҶ’ Call `confirm_delete()` before removal

### Task 2.4: Terminal Width for Tables
